About this site

This site is authored by Chris Hitchcock, a cheapskate. In early 2012, I hadn't seen Season 4 of Breaking Bad since it wasn't on Netflix yet.

After watching the cliffhanger ending of season 3 on Netflix, most people would have gone to iTunes or Amazon in order to see what happend. Instead, since season 5 was only a few months away, I decided to just check Netflix every few weeks to see if season 4 was available to stream on Netflix.

Searching Netflix every week or so to see if season 4 was available was a bit tedious, I decided there needed to be a better way. Inspired by single-serving sites like islostapeat.com, I decided to make this site.

This site is powered by a python script that checks the Netflix Instant catalog to see if season 4 of Breaking Bad is available yet. The script is run on an hourly basis in order to stay up-to-date without abusing the Netflix API. The code that drives this site is available on GitHub. It isn't anything too interesting.

Breaking Bad is now finished and all the seasons are curently availble to stream on Netflix. This site still lives. It still reports every hour if season 4 is still on Netflix.
